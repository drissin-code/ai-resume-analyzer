# AI Resume Analyzer

A web app that gives instant, AI-powered feedback on your resume — built with Streamlit and Google's Gemini API.

## What it does

Upload your resume (PDF or DOCX), enter your target job role, and get a structured analysis covering:

- Overall score (out of 10)
- Top 3 strengths
- Top 3 weaknesses
- 5 specific, actionable improvements
- Missing keywords/skills for the target role

## Tech stack

- **Frontend/UI:** Streamlit
- **AI Model:** Google Gemini API
- **File parsing:** pdfplumber (PDF), python-docx (DOCX)
